# AVERA Pilot Motion Launch Kit v0

**Date:** 2 May 2026  
**Status:** First operational launch kit for real external pilot motion  
**Purpose:** Gather the minimum document set required to begin the first real AVERA outreach-to-pilot cycle

## What This Kit Is

This kit is the first operational bundle for moving AVERA outward in a controlled way.

It is meant to connect:

- first contact
- outreach discipline
- pilot scoping
- sample export request
- first real pilot motion

It is not a product spec.

It is the working kit for the first real external cycle.

## Recommended First Motion

The first recommended motion is:

1. choose one Tier 1 target
2. send one short outreach message
3. attach the first-contact bundle
4. if interest appears, apply the pilot safety checklist before expanding scope
5. run the 10-minute adapted pilot walkthrough
6. use the intake checklist
7. request one small real sample export
8. decide whether to green-light a narrow pilot

## What This Kit Should Produce

This kit is successful if it produces:

1. one live target in motion
2. one named owner
3. one chosen artifact family
4. one sample export request
5. one follow-up scoping session
