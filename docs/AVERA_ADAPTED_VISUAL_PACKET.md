## Для чего нужен этот пакет

- иметь один чистый visual fallback, если live Streamlit shell просыпается медленно
- показать 3 adapted scenarios как единый pilot-facing набор
- сохранить product message без зависимости от текущего состояния runtime

## Рекомендуемый порядок показа

1. Открыть adapted one-pager
2. Показать HTML showcase или сохраненный screenshot
3. Пройти 10-minute pilot script
4. Завершить adapted outreach packet и вопросами про реальные артефакты design partner

## Product message

AVERA не требует идеального нового процесса.  
Она берет существующие инженерные артефакты, нормализует их и строит объяснимый risk-review контур поверх них.
