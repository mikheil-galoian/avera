## Minimum Talk Track

Say:

`AVERA is a local evidence-first review layer for engineering changes. Its current strength is that external artifacts such as verification logs, JUnit/xUnit reports, simulation CSV, and requirements exports can be normalized into one stable review path.`

Then say:

`We are not proposing a broad platform rollout first. We are proposing one narrow pilot around one artifact family your team already uses.`

## Best First Pilot Shapes

- validation/log-led regression review
- JUnit/xUnit-backed verification review
- ADAS simulation regression review

1. one artifact family
2. one representative sample export

4. one follow-up pilot scoping session
